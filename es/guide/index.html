Introducción

La Guía está diseñada para cumplir con los estándares web modernos, ser liviana y ser fácilmente accesible. Cualquiera puede sugerir mejoras o contribuir a su éxito a través de Github.

Caveat Emptor

Esta guía contiene enlaces de pago a varios sitios web de terceros. Estos enlaces ayudan a financiar la guía y permiten una experiencia sin publicidad. OfLiberty es el patrocinador principal actual de este proyecto. Para obtener una lista completa de patrocinadores, puede ver nuestra sección de patrocinadores.

Como usar esta guia

La guía sirve como un depósito de información sobre el cannabis en Pennsylvania. Aquí hay algunos trucos para usar la guía.

También acabamos de lanzar un programa de referencias para que podamos ayudar a quienes nos comparten como recurso.

Anclas de cabecera

Al lado de cada encabezado principal, verá un ícono verde cuando se sobrepase. Al hacer clic en él se agrega el ancla del encabezado a la URL. Ahora puede enlazar fácilmente a ese contenido específico en discusiones fuera de este sitio web.

Instale la guía en su teléfono

La Guía de cannabis PA es una aplicación web progresiva, lo que significa que probablemente se pueda instalar en su teléfono inteligente.

Cómo contribuir

La mayoría de las contribuciones se realizan a través del repositorio Github. Siga nuestro progreso en la página de proyectos.

Editar páginas existentes

Edite las páginas existentes. Para contribuir con el contenido, necesitará una cuenta de Github.

Navegue a nuestro repositorio. Ya sea mirar o protagonizar o ambos para mantenerse al día con los cambios.

Mientras navega por la guía, haga clic en "¡Ayúdenos a mejorar esta página!" Enlace en la parte inferior de la página. Accederá a una página que es editable.

Una vez finalizado, puede confirmar sus cambios como una nueva sucursal para su revisión.
